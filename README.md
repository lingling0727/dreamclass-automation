# 삼성드림클래스 온라인 멘토링 자동화 스크립트

자동화 스크립트를 통해 최종 생성되는 게시글 표기 양식은 아래와 같습니다.

- **제목**: [이름] 멘티 온라인 멘토링 (예: 00 멘티 온라인 멘토링)
- **해시태그**: 온라인멘토링
- **내용**: [월]월 [회차]회차
- **라이브 설명**: 늦지 않게 만나요!
- **종료 시간**: 시작 시간 기준 +29분으로 자동 계산 (예: 9시 30분 시작 시 9시 59분 종료. 따라서 다음 타임에 바로 시작해도 시간이 겹치지 않습니다.)

## 시작 전 필수 설치
라이브러리가 없는 분은 아래 명령어를 터미널에 입력하여 실행해주세요.

```bash
pip install playwright
playwright install chromium
```

## 수정해야 할 파일

### 1. config.py
나의 드림클래스 접속 아이디와 비밀번호를 입력합니다.

```python
LOGIN_ID  = "내_아이디"
LOGIN_PW  = "내_비밀번호"
```

### 2. students.json
멘티들의 이름과, 멘티를 검색할 때 쓰이는 닉네임 정보를 입력합니다.

```json
[
  { "name": "홍길동", "nickname": "홍길동123" },
  { "name": "김철수", "nickname": "철수야놀자" }
]
```

### 3. schedule.csv (스케줄 파일)
게시글을 올릴 일정을 작성합니다. 시작시간만 적으면 되며, 자동화 실행 후 작업이 완료된 일정은 다른 일정과 꼬이지 않도록 파일 내용에서 꼭 지워주세요!

```csv
이름,월,회차,일,시작시,시작분
홍길동,4,2,19,10,20
김철수,4,2,19,11,0
```

## 실행 방법

모든 파일 세팅이 끝났다면 터미널에서 아래 명령어를 실행합니다.

- **윈도우**: `python main.py`
- **맥 (Mac)**: `python3 main.py`

```bash
python main.py
```
